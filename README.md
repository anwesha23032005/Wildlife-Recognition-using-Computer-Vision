# 🐾 Wildlife-Recognition-using-Computer-Vision

A Computer Vision project that detects and classifies wildlife species using a custom-trained **YOLOv8** model. 
The system identifies animals from uploaded images and returns the detected species with confidence scores, making it suitable for wildlife monitoring and conservation applications.

---

## 📖 Overview

WildVision AI is an object detection system built using **Ultralytics YOLOv8**. The model is trained on a custom dataset to recognize the following wildlife species:

- 🐅 Tiger
- 🐘 Elephant
- 🐆 Leopard
- 🐺 Wolf
- 🐒 Primate

The project demonstrates the complete deep learning workflow, including dataset preparation, model training, inference, and API integration.

---

## ✨ Features

- Detects multiple wildlife species from images
- Real-time object detection using YOLOv8
- Returns predicted class labels with confidence scores
- Custom-trained model
- REST API integration
- Tested using Postman
- Lightweight and fast inference
